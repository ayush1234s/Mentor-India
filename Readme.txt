Mentor India - Educational Platform
Welcome to Mentor India, an innovative educational platform designed to enhance the learning experience for students. Our website allows users to explore and purchase various courses, making education more accessible and engaging.

Tech Stack
This project is built with the following technologies:

HTML: Structuring the website content.
Tailwind CSS: Creating a modern, responsive design.
JavaScript: Adding interactivity and dynamic content.
React: Building user interfaces with reusable components.
Node.js: Handling server-side logic and backend operations.
MongoDB: Storing and managing data efficiently.

Course Enrollment: Users can browse and purchase courses directly through the platform.
Responsive Design: A mobile-friendly layout ensures a seamless experience across all devices.
Dynamic Content: Interactive elements powered by JavaScript and React.
