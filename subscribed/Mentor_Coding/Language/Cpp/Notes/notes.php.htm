HTML Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create and design webpages. HTML structures a webpage and its content through a series of elements or tags.

Key Concepts:

  • Elements: Building blocks of HTML. Examples include <div>, <span>, and <a>.
  • Attributes: Provide additional information about elements. Example: <a href="url">.
  • Tags: HTML elements are wrapped in tags. Example: <h1>Heading</h1>.

Basic Structure:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document Title</title>
</head>
<body>
<h1>Hello World</h1>
</body>
</html>

Common Tags:

  • <h1> to <h6>: Header tags with varying sizes.
  • <p>: Paragraph tag.
  • <a>: Anchor tag for links.
  • <img>: Tag to display images.
